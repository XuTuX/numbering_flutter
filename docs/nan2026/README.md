Bee House는 모바일 게임이므로 App Store 링크를 사전 과제의 주 플레이 빌드로
제출한다. 별도 웹 배포는 필요하지 않다. 웹 릴리스 빌드 성공 기록은 Flutter 소스의
빌드 건전성을 확인한 내부 검증 결과일 뿐, 심사용 플레이 링크가 아니다.

## 전략 메모

Bee House의 강점은 이미 출시된 완성도와 검증 가능한 개발 이력이다. 사전 과제에서는
실시간 생성형 AI 기능을 억지로 추가하기보다, 사람이 규칙과 품질 기준을 정하고 AI가
분석, 구현, 리팩터링, 회귀 테스트를 가속한 `AI-directed production` 과정을 정직하게
보여주는 편이 현재 결과물과 가장 잘 맞는다. 런타임에는 결정론적 게임 규칙을 유지해
공정한 랭킹과 재현 가능한 리플레이를 보장한다.
