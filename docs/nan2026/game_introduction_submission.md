# Bee House(벌집게임) - 게임 소개 및 설명

## 1. 게임 제목 및 한 줄 소개

- 게임 제목: **Bee House(벌집게임)**
- 장르: 타임어택 육각 경로 퍼즐
- 지원 플랫폼: iPhone, iPad
- 한 줄 소개: **상단의 색 순서를 읽고, 육각 보드 위에서 같은 순서의 타일을 연결해 최고 기록에 도전하는 60초 드래그 퍼즐.**

Bee House는 화면 상단의 컬러 바에 표시된 연속 색 구간을 확인한 뒤, 같은 순서의 육각 타일을 보드에서 찾아 한 번의 드래그로 연결하는 퍼즐 게임이다. 같은 색을 단순히 모으는 방식이 아니라 색의 순서와 타일의 인접 관계를 함께 판단해야 하며, 긴 경로와 빠른 연속 성공을 통해 더 높은 점수와 콤보를 얻을 수 있다.

## 2. 게임 방법

### 목표

제한 시간 안에 컬러 바와 일치하는 경로를 최대한 많이 완성해 높은 점수를 얻는 것이 목표다. 짧은 경로를 빠르게 연결하거나, 더 큰 점수와 시간 보너스를 주는 긴 경로를 탐색하는 전략을 선택할 수 있다.

### 조작

1. 화면 상단의 10칸 컬러 바에서 길이 3 이상의 연속 색 구간을 찾는다.
2. 보드에서 첫 번째 색과 같은 육각 타일을 누른다.
3. 손가락을 떼지 않고 서로 인접한 타일을 컬러 바와 같은 순서로 드래그한다.
4. 경로를 완성한 뒤 손가락을 떼면 매칭에 성공한다.
5. 성공한 타일과 컬러 바 구간은 새 색으로 채워지며, 점수와 추가 시간이 주어진다.

### 핵심 규칙

- 최소 3개의 타일을 연결해야 한다.
- 육각형 기준으로 서로 맞닿아 있는 타일만 연결할 수 있다.
- 한 번의 드래그에서 같은 타일을 두 번 사용할 수 없다.
- 색의 구성뿐 아니라 순서도 컬러 바의 연속 구간과 정확히 같아야 한다.
- 긴 경로와 4초 이내의 연속 성공은 더 높은 점수와 콤보를 제공한다.

### 종료 조건

다음 중 하나가 발생하면 게임이 종료된다.

1. 남은 시간이 0초가 된 경우
2. 현재 보드에서 컬러 바와 일치하는 유효한 경로를 더 이상 만들 수 없는 경우

게임 종료 후에는 최종 점수와 최고 기록을 확인하고, 플레이를 리플레이하거나 결과를 공유할 수 있다.

## 3. 실행 방법

### 심사용 권장 실행 방법

1. 아래 App Store 링크를 iPhone 또는 iPad에서 연다.
2. **받기** 버튼을 눌러 앱을 무료로 설치한다.
3. 설치가 완료되면 **Bee House**를 실행한다.
4. 홈 화면의 플레이 버튼을 눌러 일반 게임을 시작한다.
5. 일반 플레이는 로그인 없이 이용할 수 있으며, 온라인 랭킹과 오늘의 퍼즐 공식 도전 등 일부 기능은 로그인이 필요할 수 있다.

### 소스 코드 실행 방법

1. Flutter SDK를 설치한다.
2. GitHub 저장소를 clone한다.
3. `SUPABASE_URL`, `SUPABASE_ANON_KEY`를 `--dart-define`으로 설정한다.
4. 프로젝트 폴더에서 `flutter pub get`을 실행한다.
5. 연결된 기기 또는 시뮬레이터에서 아래 명령으로 실행한다.

```bash
flutter run \
  --dart-define=SUPABASE_URL=... \
  --dart-define=SUPABASE_ANON_KEY=...
```
