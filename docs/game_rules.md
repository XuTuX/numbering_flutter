# NUMBERING 게임 핵심 규칙 가이드

이 문서는 NUMBERING 게임의 보드 구성, 매칭 로직, 점수 및 시간 시스템에 대한 핵심 규칙을 정의합니다. 본 규칙을 수정할 때는 반드시 사용자의 허락을 받아야 합니다.

## 1. 보드 및 구성 요소
- **보드 크기**: 7행 6열의 육각형(Hexagon) 타일로 구성됩니다.
- **컬러 바 (Color Bar)**: 화면 상단에 10개의 색상 타일이 흐름(Stream) 형태로 배치됩니다.
- **색상 종류**: Coral, Amber, Mint, Azure, Violet 총 5가지 색상을 사용합니다.
- **무지개 블록**: 한 번의 리필에서는 무지개 블록이 최대 1개만 등장합니다. 등장 확률은 현재 보드에 남아 있는 무지개 블록 수를 기준으로 하며, 0개면 **10%**, 1개면 **5%**, 2개면 **3%**, 3개 이상이면 **1%**입니다.

## 2. 매칭 및 드래그 규칙
- **매칭 조건**:
    - 인접한 육각 타일을 드래그하여 연결합니다.
    - 연결된 타일들의 색상 순서가 상단 **컬러 바의 연속된 구간**과 정확히 일치해야 합니다.
    - 최소 매칭 길이는 **3개 타일**입니다.
- **상태 변화**:
    - 매칭에 성공한 구간은 짧게 빛나며 축소되어 사라지고, 새로운 타일이 위에서 떨어져 빈자리를 채웁니다.
    - 컬러 바에서 사용된 구간은 제거되고 새로운 색상으로 뒤가 채워집니다.

## 3. 점수 시스템 (Scoring)
- **기본 점수**: 매칭 성공 시 기본 100점이 부여됩니다.
- **길이 보너스**: 3개를 초과하는 타일(`n = 길이 - 3`)에 대해 추가 점수가 부여됩니다.
    - 식: `(n * 150) + (n^2 * 50)`
- **콤보 보너스**: 이전 매칭 후 4초 이내에 다음 매칭을 성공하면 콤보가 쌓입니다.
    - 식: `(콤보 횟수 - 1) * 50`

## 4. 시간 시스템 (Time & Combo)
- **제한 시간**: 기본 60초로 시작합니다.
- **시간 보너스**: 매칭 성공 시 길이에 따라 시간이 추가됩니다. (최대 25초까지 누적 가능)
    - 식: `1.4 + (길이 - 2) * 0.65` 초
- **콤보 판정**: 마지막 매칭 성공 시점으로부터 **4초 이내**에 다음 매칭이 완료되어야 콤보로 인정됩니다.
- **콤보 표시**: 컬러 바 상단의 `COMBO` 옆 원형 타이머가 정상 매칭 직후 4초 동안 줄어듭니다. 원 안에는 현재 콤보 수를 표시하고, 2초 이하에서는 오렌지, 0.8초 이하에서는 코랄색으로 바뀝니다.

## 5. 허니 러시 (일반 모드)
- **발동**: 일반 모드에서 컬러 바 전체와 일치하는 10칸 정상 매칭을 성공하면 0.1초 안에 즉시 발동합니다. 허니 러시 입력과 기존 매치 제거·보드 리필 애니메이션은 동시에 진행합니다.
- **지속 시간**: 4초이며 일반 제한 시간과 기존 콤보 시간은 계속 감소합니다.
- **입력**: 인접한 미수집 타일을 2칸 이상 자유롭게 연결할 수 있고, 4초 동안 여러 경로를 만들 수 있습니다.
- **중복 방지 및 연출**: 한 허니 러시에서 점수를 받은 타일은 다시 점수를 주지 않습니다. 드래그를 놓으면 타일이 빠르게 축소·페이드되고, 러시 종료 시 원래 보드로 복귀합니다.
- **점수**: 수집 타일당 25점입니다. 시간 회복, 컬러 바 소비, 일반 매치 수, 최장 경로, 일반 콤보에는 포함하지 않습니다.
- **보드·컬러 바 보존**: 허니 러시 중에도 컬러 바의 색상과 패널 높이를 유지하며, 패널 배경만 노란 러시 상태로 전환합니다. 허니 러시 드래그는 실제 보드 색과 컬러 바 데이터를 변경하지 않습니다.
- **적용 범위**: 서버 검증이 분리된 HTR4를 사용하는 일반 모드에만 적용합니다. 오늘의 퍼즐 공식 도전·연습·튜토리얼에는 적용하지 않습니다.

## 6. 게임 종료 조건
- **시간 초과**: 남은 시간이 0초가 되면 종료됩니다.
- **수 부족 (No Moves)**: 보드 내에서 컬러 바와 일치하는 경로를 더 이상 만들 수 없을 때 종료됩니다. (매칭 성공 후 항상 체크)

## 7. 확인창 및 이탈
- **시간 유지**: 재시작·나가기 확인창이 열려 있어도 제한 시간과 콤보 시간은 계속 흐릅니다.
- **일반 모드 재시작**: 확인 후 현재 점수와 진행 내용을 버리고 새 시드로 시작합니다.
- **오늘의 퍼즐**: 공식 일일 도전에서는 재시작을 제공하지 않습니다. 중도 이탈 시 현재 기록은 제출하지 않습니다.
- **앱 백그라운드 전환**: 백그라운드에서도 기존 마감 시간을 유지합니다.

## 8. 공식 점수 검증
- 전체·주간·오늘 랭킹 점수는 시드와 `HTR3` 리플레이를 서버에서 처음부터 재계산한 결과만 저장합니다.
- 로그인 시 로컬 최고점을 서버 랭킹으로 승격하지 않습니다. 로컬 기록은 기기 기록으로 유지되고, 이후 검증된 플레이가 서버 기록을 갱신합니다.
- 서버 재계산 결과가 제출 점수, 매치 수, 최대 콤보, 최장 경로 또는 종료 시간과 다르면 제출을 거절합니다.
- 일반 모드에서 허니 러시가 발생한 기록은 `HTR4`로 제출하며, 서버는 10칸 발동·4초 입력 창·인접성·타일 중복 수집·타일당 25점을 검증합니다. 일일 공식 점수에는 `HTR4`를 허용하지 않습니다.
