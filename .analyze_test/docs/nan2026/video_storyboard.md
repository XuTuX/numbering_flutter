# 45초 실제 플레이 영상 구성표

공모전 규정상 AI 조작·합성 없이 실제 게임 플레이 화면 그대로 녹화한다. 세로 화면
녹화를 권장하며, 편집은 앞뒤 무음 제거와 자막 추가 정도로 제한한다.

| 구간 | 실제 화면 | 짧은 자막 |
| --- | --- | --- |
| 0~3초 | 홈에서 Bee House 로고와 플레이 버튼 | 색의 흐름을 읽는 60초 육각 퍼즐 |
| 3~9초 | 게임 시작, 상단 컬러 바와 보드 | 연속된 색 순서를 찾고 |
| 9~17초 | 길이 3~4 경로 드래그 성공 | 같은 순서로 인접 타일을 드래그 |
| 17~25초 | 더 긴 경로와 점수/시간 증가 | 길수록 더 큰 점수와 시간 보너스 |
| 25~31초 | 빠른 연속 성공과 콤보 | 4초 안에 이어서 콤보 |
| 31~37초 | 오늘의 퍼즐 또는 달력 | 모두가 같은 시드로 경쟁 |
| 37~42초 | 결과 화면과 리플레이 | 기록을 그대로 다시 보기 |
| 42~45초 | 랭킹 또는 홈 | Bee House - 지금 플레이하세요 |

## 업로드 설정

- 길이: 30~60초, 권장 45초
- 공개 범위: 공개 또는 일부 공개
- 제목 예시: `[NAN 2026] Bee House 실제 플레이`
- 설명란: App Store 링크와 GitHub 링크
- 업로드 후 URL을 `game_introduction.md`와 게임 소개 PDF에 반영
